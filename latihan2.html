font style

CSS dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa porperty yang dapat digunakan untuk mengatur style teks teks yaitu sebagai berikut.

  • text-align: digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, right.
  • color: digunakan untuk memberikan warna pada teks.
  • letter-spacing: mengatur jarak antar karakter.
  • word-spacing: mengatur jarak kata.
